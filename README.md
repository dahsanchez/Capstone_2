## EmporiUm Sales Analysis
### This project explores sales data for EmporiUm, a student‑focused bookstore chain offering books, tech products, and art supplies through in‑store and online channels. Using Python for data analysis, I answered sales management questions by region and territory, importing csv files into my Juypter notebook, and created a plot charts using matplotlib to support insights shared in a demo and live presentation. The sales territories I have are Florida and Texas.
